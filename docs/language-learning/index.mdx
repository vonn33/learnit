# Language Learning

This project synthesises two complementary books on language learning into a navigable reference:

| Source | What it is |
|---|---|
| **Fluent Forever** — Gabriel Wyner (2014) | A complete, practical method built on cognitive science: pronunciation → vocabulary → grammar → fluency via spaced repetition flash cards |
| **What Do You Need to Know to Learn a Foreign Language?** — Paul Nation (2014) | A research-grounded framework: four balanced learning strands, vocabulary thresholds, and the conditions that make any method work |
| **Two Maps, One Territory** (composite report) | A full comparative analysis of both books — convergences, divergences, genuine contradictions, a synthesised integrated framework, and critical evaluation |

## How to Navigate

**Quick References** — One-page condensed guides to each book. Start here if you want the core ideas fast.

**Two Maps, One Territory** — The full comparative analysis. Read front-to-back once, then use as a lookup. Structured as:
- **Part 0** — Orientation: what each book is, side-by-side at a glance
- **Part I** — Each book in its own right: core system, evidence, what it gets right, where it's thin
- **Part II** — Head-to-head: where they agree, diverge, and genuinely contradict
- **Part III** — The synthesised view: master principles and an integrated daily framework
- **Part IV** — Critical evaluation: shared blind spots, post-2014 research updates
- **Appendix** — Key tools, FSI time data, researcher reference
